# Syllabus Authoring

- Syllabus edits MUST preserve the course's actual taught sequence and current progress.
- When adjusting a syllabus to actual progress, use concrete lesson numbers, textbook pages, material files, or user-provided progress facts.
- Do not infer actual progress from general course knowledge when the user provides specific progress information.
- If a syllabus change shifts assessment scope, report the affected quiz/exam/course-docs repositories so they can be updated in a separate task.
- Syllabus entries SHOULD describe lesson scope in stable material terms rather than one-off classroom events.
- Do not add a separate material `type` field unless the user explicitly requests one.
