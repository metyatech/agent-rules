# Course Exam Retake Authoring

- Retake questions MUST preserve the original exam's learning target, question
  type, scoring rubric, and expected difficulty band.
- Retake questions MUST be materially different from both the original exam
  question and any preparation question while remaining gradable by the same
  criteria.
- If a preparation question marks the allowed variation scope, the retake
  question MUST vary only within that scope.
- Retake authoring MUST keep operational file/layout rules in the course exam
  repository or prompt; this domain defines only reusable authoring constraints.
