# Course Exam Retake Authoring

- Retake questions MUST preserve the original exam's learning target, question
  type, `## Scoring` bullet list, and expected difficulty band.
- Retake questions MUST use the same `## Scoring` bullet list as the original
  exam question.
- Retake questions MUST be materially different from both the original exam
  question and any preparation question while remaining gradable by the same
  criteria.
- The retake assessment manifest `points` MAY differ from the original exam's
  `points`; the `## Scoring` criterion text and its ordering MUST still align
  one-to-one with the original.
- If a preparation question marks the allowed variation scope, the retake
  question MUST vary only within that scope.
- Retake authoring MUST keep repository layout and workflow rules aligned with the `course-exams` domain rules.
