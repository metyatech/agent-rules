# Contributing to Agent Rules

Thank you for your interest in contributing!

## How to Contribute

1.  Fork the repository.
2.  Create a branch for your changes.
3.  Add or modify rules in `rules/global/` or `rules/domains/`.
4.  Ensure your rules follow the established style and structure.
5.  Open a Pull Request.

## Rules Composition

`AGENTS.md` is a generated file. Do not edit it directly.
Instead, update the source rule modules and run `compose-agentsmd` to verify.
